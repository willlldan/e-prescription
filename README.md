## Installation

```
We assumed you have installed Laravel

1. Prepare database
    - database name: e_prescription
2. Copy file env.example and rename to .env
3. In terminal, run this code
    - composer install
    - php artisan key:generate
    - php artisan migrate:fresh --seed
    - php artisan serve
4. Application already running
```
